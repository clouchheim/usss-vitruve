# USSS Vitruve x Teamworks AMS Integration

Integration to connect Vitruve VBT Technology and USSS Teamworks AMS.  

## Requires
- API_KEY = api key from vitruve.fit Professional Subscription
- A360_USER = username for Teamworks AMS upload profile
- A360_PASSWORD = password for Teamworks AMS upload profile

Notes: the Teamworks AMS user profile needs to have coach access to all athletes it will be writing data for as well as the 'Vitruve VBT' form. This profile cannot have MFA enabled
